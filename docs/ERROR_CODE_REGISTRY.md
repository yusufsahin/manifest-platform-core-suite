# ERROR CODE REGISTRY v1.0

All produced errors MUST use codes in this registry. Unknown codes MUST fail conformance.

## Parse

- E_PARSE_SYNTAX
- E_PARSE_INVALID_TOKEN
- E_PARSE_UNSUPPORTED_FORMAT

## Meta

- E_META_UNKNOWN_KIND
- E_META_MISSING_REQUIRED_FIELD
- E_META_TYPE_NOT_ALLOWED
- E_META_FUNCTION_NOT_ALLOWED
- E_META_BREAKING_CHANGE

- E_VALID_DUPLICATE_DEF
- E_VALID_UNRESOLVED_REF
- E_VALID_CYCLE_DETECTED
- E_VALID_NAMESPACE_CONFLICT
- E_VALID_INVALID_WORKFLOW

## Reason Codes (R\_\*)

Reason codes appear in `Decision.reasons[].code`. They explain WHY a decision was made.
Unknown `R_*` codes MUST fail conformance, same as `E_*` codes.

### Policy Reasons

- R_POLICY_ALLOW
- R_POLICY_DENY

### ACL Reasons

- R_ACL_ALLOW_ROLE
- R_ACL_DENY_ROLE
- R_ACL_ALLOW_ABAC
- R_ACL_DENY_ABAC

### Workflow Reasons

- R_WF_GUARD_PASS
- R_WF_GUARD_FAIL
- R_WF_AUTH_DENIED

### Governance Reasons

- R_GOV_SIGNATURE_VALID
- R_GOV_ATTESTATION_PASSED
